# VeteranMeet – Web Engineering Project


## Project Overview
VeteranMeet is a full-stack web application designed to connect retired professionals (veterans) with community service opportunities and organizations.


This project is developed as part of the **Web Engineering – Fall 2025** course.


## Tech Stack
- Frontend: React (Vite) + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT
